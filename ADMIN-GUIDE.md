# Руководство администратора MARSIANS MCC

## Как добавить новое мероприятие

1. Откройте файл `data/events.json`
2. Добавьте новый объект в массив:

```json
{
  "id": 4,
  "title": "Название мероприятия",
  "date": "2025-01-15",
  "description": "Описание мероприятия. Можно несколько предложений.",
  "image": "images/events/event4.jpg",
  "category": "Гонки"
}
```

3. Загрузите фото мероприятия в папку `images/events/`
4. Закоммитьте изменения в GitHub:

```bash
git add data/events.json images/events/
git commit -m "Добавлено мероприятие: Название"
git push
```

### Категории мероприятий:
- **Гонки** - соревнования, ралли, рейды
- **Тренировка** - тренировочные заезды
- **Встреча** - клубные встречи
- **Поездка** - групповые поездки
- **Другое** - прочие мероприятия

## Как добавить нового члена клуба

1. Откройте файл `data/members.json`
2. Добавьте новый объект:

```json
{
  "id": 11,
  "name": "Имя Фамилия",
  "role": "MEMBER",
  "title": "Член клуба",
  "achievements": "Краткое описание достижений",
  "image": "images/members/member11.jpg",
  "bike": "Название мотоцикла"
}
```

3. Загрузите фото (желательно квадратное, 400x400px) в папку `images/members/`
4. Закоммитьте:

```bash
git add data/members.json images/members/
git commit -m "Добавлен член клуба: Имя Фамилия"
git push
```

### Роли членов:
- **FIRST5** - руководство клуба (выделяется белой рамкой)
- **MEMBER** - обычные члены клуба

## Как загрузить фотографии

### Вариант 1: Через GitHub (простой)

1. Перейдите на https://github.com/igorgorbenko/marsians
2. Откройте папку `images/events/` или `images/members/`
3. Нажмите **Add file** → **Upload files**
4. Перетащите фотографии
5. Введите описание коммита и нажмите **Commit changes**

### Вариант 2: Через командную строку

```bash
# Скопируйте фото в нужную папку
cp ~/Desktop/photo.jpg images/events/event4.jpg

# Закоммитьте
git add images/
git commit -m "Добавлены фотографии"
git push
```

### Рекомендации по фото:

**Для мероприятий:**
- Размер: 800x600px (горизонтальная ориентация)
- Формат: JPG
- Вес: до 500KB

**Для членов клуба:**
- Размер: 400x400px (квадрат)
- Формат: JPG
- Вес: до 300KB

## Использование Cloudinary (рекомендуется)

Для оптимального хранения и быстрой загрузки фото рекомендую использовать Cloudinary:

1. Зарегистрируйтесь на https://cloudinary.com (бесплатно 25GB)
2. Загрузите фото через их интерфейс
3. Скопируйте URL загруженного фото
4. Используйте этот URL в JSON файлах:

```json
{
  "image": "https://res.cloudinary.com/your-account/image/upload/v123456/event1.jpg"
}
```

**Преимущества:**
- Автоматическая оптимизация изображений
- CDN для быстрой загрузки
- Не увеличивает размер репозитория GitHub

## Редактирование контента

### Изменить описание мероприятия

1. Откройте `data/events.json`
2. Найдите нужное мероприятие по `id`
3. Измените поля `title`, `description` или другие
4. Сохраните и закоммитьте

### Удалить мероприятие

1. Удалите объект из массива в `data/events.json`
2. Закоммитьте изменения

### Изменить информацию о члене клуба

1. Откройте `data/members.json`
2. Найдите нужного члена по `id`
3. Измените данные
4. Сохраните и закоммитьте

---

## Публикация изменений

После любых изменений в файлах `data/` или `images/`:

```bash
git add .
git commit -m "Описание изменений"
git push
```

GitHub Pages автоматически обновится через 1-2 минуты.

## Частые вопросы

**Q: Как отредактировать JSON файл?**
A: Можно использовать любой текстовый редактор или редактировать прямо на GitHub.

**Q: Сколько мероприятий можно добавить?**
A: Неограниченно. Они отображаются в порядке от новых к старым.

**Q: Как изменить порядок отображения членов?**
A: FIRST5 всегда отображаются первыми. Порядок внутри групп - как в JSON файле.

**Q: Что делать, если сайт не обновился?**
A: Подождите 2-3 минуты и обновите страницу с очисткой кэша (Ctrl+F5 или Cmd+Shift+R).
