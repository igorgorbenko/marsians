# История изменений

## Версия 2.0 - Черно-белый дизайн с логотипом (31.12.2024)

### Основные изменения:

#### 1. Полная смена цветовой схемы на черно-белую
- **Фон:** Чистый черный (#000000) вместо темно-серого
- **Акцентный цвет:** Белый (#ffffff) вместо оранжево-красного (#ff3c00)
- **Границы:** Белые вместо серых
- **Эффекты:** Белое свечение вместо красного

#### 2. Интеграция оригинального логотипа MARSIANS MCC
- Скачан официальный логотип с сайта motorallycup.ru
- Логотип добавлен в трех местах:
  - **Header** - компактная версия (50px)
  - **Hero-секция** - большой логотип (400px) с плавающей анимацией
  - **Footer** - средний размер (80px)

#### 3. Анализ логотипа MARSIANS MCC:
**Элементы дизайна:**
- Круговая композиция с текстом "MARSIANS" и "NOMAD"
- V-twin двигатель мотоцикла в центре
- Символ пламени внутри двигателя
- Две молнии по бокам
- Прямоугольные таблички "MRC" и "MCC"
- Футуристичный шрифт в стиле космических программ

**Цвета:**
- Полностью черно-белая палитра
- Высокий контраст без оттенков серого
- Строгий, брутальный стиль

#### 4. Обновленные CSS-переменные:
```css
--primary-bg: #000000       (было #0a0a0a)
--secondary-bg: #0a0a0a     (было #1a1a1a)
--accent-color: #ffffff     (было #ff3c00)
--border-color: #ffffff     (было #333333)
```

#### 5. Новые анимации:
- **logoFloat** - плавающий логотип в hero-секции
- Обновлен эффект **textGlow** с белым свечением
- Drop-shadow для логотипа

#### 6. Адаптивный дизайн для логотипа:
- Desktop: hero-logo 400px, header 50px, footer 80px
- Mobile: hero-logo 250px, header 35px, footer 60px

### Технические детали:

**Новые файлы:**

Сайт теперь полностью соответствует фирменному стилю MARSIANS MCC:
- ✅ Черно-белая гамма как в оригинальном логотипе
- ✅ Строгий, брутальный дизайн
