# MARSIANS MCC - Официальный сайт мотоклуба

Информационный сайт мотоклуба MARSIANS MCC с современным, брутальным дизайном.

## Описание

Одностраничный сайт для мотоклуба спортсменов MARSIANS MCC, основанного в 2019 году. Сайт содержит информацию о клубе, целях, членстве, географии и контактах.

## Особенности дизайна

- Темная цветовая схема с акцентом на красный цвет
- Брутальный, строгий стиль
- **Марсианский шрифт NASALIZATION** (вдохновлен логотипом NASA 1975)
- Современные анимации и переходы
- Полностью адаптивный дизайн для всех устройств
- Плавная прокрутка между секциями
- Эффекты при наведении на элементы

## Запуск сайта

### Вариант 1: Открыть файл напрямую
Просто откройте файл `index.html` в любом современном браузере (Chrome, Firefox, Safari, Edge).

### Вариант 2: Локальный сервер (рекомендуется)

**С помощью Python:**
```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000
```

**С помощью Node.js (http-server):**
```bash
npx http-server -p 8000
```

**С помощью PHP:**
```bash
php -S localhost:8000
```

После запуска откройте браузер по адресу: `http://localhost:8000`

## Настройка

### Изменение цветов
Основные цвета определены в CSS переменных в файле `css/style.css`:

```css
:root {
    --primary-bg: #0a0a0a;        /* Основной фон */
    --secondary-bg: #1a1a1a;      /* Вторичный фон */
    --accent-color: #ff3c00;       /* Акцентный цвет */
    --text-primary: #ffffff;       /* Основной текст */
    --text-secondary: #cccccc;     /* Вторичный текст */
}
```

### Добавление изображений
- Разместите изображения в папке `images/`
- Для hero-секции можно добавить фоновое изображение в `css/style.css` в классе `.hero`
- Рекомендуется добавить логотип клуба
